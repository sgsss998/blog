Gemini CLI 是 Google 的一款命令行工具，可以帮你修改文件、改代码、执行操作。是根植于终端命令行里面的 agent 助手。

只要你有谷歌账号，就可以免费使用它，gemini cli 每天都有不少的免费额度，用不完。智能还不错。

可以作为 AI 入门的必装软件。

## 安装步骤

安装 gemini cli 非常简单：

### 1. 复制安装命令

登录这个网址：https://geminicli.com/

然后复制其中的命令：

```bash
npm install -g @google/gemini-cli
```

### 2. 打开终端

很多人不知道终端是什么，也不知道怎么打开终端，终端其实就是一个窗口，能够直接输入命令给计算机。

**mac 打开终端的方法：**

同时按住 "command+空格"，然后输入"终端"或者"terminal"，然后回车。然后就能打开终端了。

Windows 系统请自行百度。

### 3. 执行安装命令

在终端里面复制命令：

```bash
npm install -g @google/gemini-cli
```

然后回车。gemini cli 就装好了。

### 4. 启动

打开终端，在终端里面输入：

```bash
gemini
```

然后回车。

然后就会弹出让你绑定 Google 账号的网页，登录，绑定，即可。

然后你就可以正式使用 gemini cli 了。
