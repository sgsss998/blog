这是一个 Claude Code skill，专门用来做网页版演示文稿。核心特点：

- 用 AI 生成自带动画效果的 HTML 幻灯片，可以从零创建，也可以把 PPT 文件一键转成网页版。
- PPT 转换：把 .pptx 转成带动画的 HTML，保留图片和内容。
- 视觉风格选择：AI 生成多种风格预览供挑选。
- 零依赖：纯 HTML（CSS/JS 内联），不需要 npm、不需要构建工具。
- 防 AI 味：精选独特风格。
- 生产可用：代码规范、有注释、可直接改。

**安装方式**

```text
git clone https://github.com/zarazhangrui/frontend-slides.git ~/.claude/skills/frontend-slides
```

装好后在 Claude Code 里输入 `/frontend-slides` 即可使用。

我用它做过 OpenClaw 演讲稿和护肤类教程，效果很稳。免费 skill，需在 Claude Code 中使用。
