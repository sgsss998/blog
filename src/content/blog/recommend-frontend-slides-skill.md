这是一个 Claude Code skill，专门用来做网页版演示文稿。核心特点：

用 AI 生成自带动画效果的 HTML 幻灯片，可以从零创建，也可以把 PPT 文件一键转成网页版。

PPT 转换 把 .pptx 文件转成带动画的 HTML 演示文稿，保留所有图片和内容

视觉风格选择 不需要用语言描述审美偏好，AI 生成3种风格预览图让你选

零依赖 纯 HTML 文件（CSS/JS 内联），不需要 npm、不需要构建工具

防 AI 味 精选独特风格，拒绝千篇一律的紫白渐变

生产可用 代码规范、有注释、可直接改

安装方式

git clone https://github.com/zarazhangrui/frontend-slides.git ~/.claude/skills/frontend-slides

装好后 Claude Code 里输入 /frontend-slides 即可使用。
